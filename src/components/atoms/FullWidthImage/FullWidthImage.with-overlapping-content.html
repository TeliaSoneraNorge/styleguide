If you add the --with-content-overlap CSS class then the next piece of content will overlap the image.

It's recommended to wrap the overlapping content in a <div class="main-content"> to make it responsive and <section class="body-text-container"> to give the content a max-width to restrict the line length in the text for readability (accessibility) reasons.

If you also specify the main-content--padding-top CSS class the main-content element will be padded on top. This is solved with a modifier class because not all elements within the main-content element should have padding on top.

If you specify the main-content--padded-sides CSS class the overlapping content will look the same on smaller devices until it wraps beneath the image on the smallest devices.

Add main-content--bg-color-white if you want the background-color to be white.

It's also recommended to keep the height of the next piece of content tall enough to cover the mask.
